# surgitrack
SurgiTrack Lite - A simple surgery case logbook and management system built with Streamlit for surgeons to record, manage, and export surgery records.
Features:
✅ Add Surgery Cases
✅ View Surgery Logbook
✅ Search Patient Records
✅ Dashboard Analytics
✅ Excel Export
✅ Lightweight and Easy to Deploy

Technology Stack:
- Python
- Streamlit
- Pandas
- OpenPyXL

Ideal for surgeons, medical students, and healthcare professionals who want a simple digital surgery case tracking system.
